# House Seeker 🏠🔍

A multiplayer hide-and-seek Roblox game built inside a real-world New Zealand house floor plan.

2–8 players. One Seeker. Multiple rooms to hide in. 3 minutes to survive.

## Features
- Multi-room house map imported from a real NZ property scan
- Role-based gameplay: Seeker vs Hiders
- Auto round management with lobby system
- Custom in-game HUD with countdown timer
- Power-ups: smoke bombs, speed boost, decoys (coming soon)

## Built With
- Roblox Studio
- Luau scripting language
- Real architectural floor plan (Auckland, New Zealand)

## How to Play
1. Join the game on Roblox
2. Wait for 2+ players in lobby
3. One player is randomly assigned as Seeker
4. Hiders have 30 seconds to find a hiding spot
5. Seeker must find everyone before time runs out
